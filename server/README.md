# Server
## Dependencies
- gradle
- java-17-openjdk
## How To Run
- `cd` into this directory
### Linux
- `sudo chmod +x gradlew`
- `./gradlew bootRun` 
### Windows
- `gradlew.bat`
## How To Access
After starting the server in the IDE, the server will be hosted at [https://localhost:8080](https://localhost:8080)
